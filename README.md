# Students-Social-Media-Addiction-
This project analyzes social media addiction among students and its impact on mental health, sleep patterns, academic performance, and daily behavior. The dashboard was built using Power BI with a focus on clean design, interactivity, and storytelling.
Dataset Description

The dataset contains student-level survey data with the following key fields:

Age

Gender

Country

Academic_Level

Most_Used_Platform

Avg_Daily_Usage_Hours

Sleep_Hours_Per_Night

Mental_Health_Score

Affects_Academic_Performance

Relationship_Status

Addicted_Score

A calculated column was created:

Addiction_Level (Low / Medium / High)

 Tools & Technologies

Power BI Desktop

Power Query (data cleaning & transformation)

DAX (measures and KPIs)

Dashboards Structure
The project is divided into two dashboards:

Dashboard 1 – Overview: High-level insights and global patterns

Dashboard 2 – Behavior & Impact: Deep-dive analysis into behavior, platforms, and impacts
Higher addiction scores are associated with lower mental health scores

Students with high addiction tend to sleep fewer hours

Certain platforms show significantly higher addiction levels
